Project One

A web application built using HTML, CSS, and JavaScript This project involves the development of a To-Do List Web Application using HTML, CSS, and JavaScript. The application allows users to add, mark as completed, and delete tasks, providing an intuitive way to manage their daily activities. The user interface is designed using HTML and styled with CSS, ensuring a clean and visually appealing layout. JavaScript is used to handle the dynamic functionality, including the ability to add tasks, toggle task completion, and remove tasks. The tasks are stored in the browser’s local storage, ensuring that they persist even after the page is refreshed. This project demonstrates how to integrate these core web technologies to create a fully functional, interactive web application. It also provides a solid foundation for exploring more advanced features, such as due dates, task prioritization, and persistent data storage, allowing for further expansion as needed. .

A responsive portfolio website built with React and CSS This project involves the creation of a **responsive portfolio website** built with **React** and **CSS**. The website is designed to showcase personal projects, skills, and professional experience in a visually appealing and user-friendly manner. By leveraging React, the site features a dynamic, component-based structure that makes it easy to manage and update content.
